### How to run (Setup)


1. **Prerequisites:**

    - Java 21 (sdkman)
    - Maven (sdkman)
    - Postman
    - curl
    - git


2. **Clone the repository:**

   ```bash
   git clone https://github.com/j-imsa/AdlasApp.git
   cd AdlasApp-master
   ```


3. **Build the project:**

   ```bash
   mvn clean install
   ```


4. **Run the app:**

   ```bash
   cd target
   java -jar AdlasApp-0.0.1.jar
   ```
